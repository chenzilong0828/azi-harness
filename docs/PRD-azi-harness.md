# PRD 入口

本文件是 `docs/` 目录下的 PRD 入口，方便在文档中心内集中查看。

当前完整 PRD 正文仍在项目根目录：

- [../PRD-azi-harness.md](../PRD-azi-harness.md)

建议阅读顺序：

1. 先读 [MEMORY.md](./MEMORY.md)，快速了解当前完成度、关键决策和接手上下文。
2. 再读根目录完整 PRD，获取产品目标、阶段计划、规格要求和风险清单。
3. 如需对照实现，再查看：
   - [architecture.md](./architecture.md)
   - [cli.md](./cli.md)
   - [runtime-protocol.md](./runtime-protocol.md)

之所以保留根目录 PRD，是为了避免维护两份超长正文产生漂移；`docs/` 目录负责集中入口和交接索引。
